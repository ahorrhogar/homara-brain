# Tracking de clicks

## Datos por evento
- Timestamp.
- URL origen (artículo / categoría / ficha).
- Producto.
- Partner.
- User agent + país.
- Sesión (anonimizada).

## Usos
- KPIs de conversión por pieza.
- Optimización de CTAs.
- Detección de enlaces rotos (clicks 0 sostenidos en producto activo = revisar).
- Atribución cruzada con comisiones reportadas por programa.
