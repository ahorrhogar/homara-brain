# Estructura de categorías

## Principios
- Jerarquía de **dos o tres niveles máximo**: `Categoría → Subcategoría → (filtro/uso)`.
- Cada nodo tiene **texto SEO largo único** (sin copiar a competencia).
- Una URL por intención.
- Breadcrumbs siempre.
