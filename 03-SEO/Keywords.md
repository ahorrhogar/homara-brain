# Keywords

## Tipos a trabajar

- **Comerciales** ("comprar X", "X barato", "mejor X")
- **Comparativas** ("X vs Y", "diferencias entre X y Y")
- **Long tail informacionales** ("cómo elegir X para Y")
- **Marca + producto** (cuando hay interés real, evitando trademarks)
- **Estacionales** (ver [[Contenido-estacional]])

## Workflow

1. Investigación inicial por **cluster** (no keyword aislada).
2. Definir **página principal** del cluster (la cabeza).
3. Soportar con **artículos satélite** y enlazar a la cabeza.
4. Trackear posición y revisar a 30 / 60 / 90 días.
