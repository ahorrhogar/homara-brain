# Checklist de publicación

Antes de pulsar "publicar":

## Contenido
- [ ] Cumple [[Estilo-editorial|estilo editorial]] y [[../01-Empresa/Tono-y-voz|tono]].
- [ ] Verificación factual hecha (precios, medidas, materiales).
- [ ] Sin frases que suenen a IA.
- [ ] CTAs claros pero no agresivos.

## SEO
- [ ] Keyword principal en H1, primer párrafo, slug, meta title, meta description.
- [ ] Headings jerárquicos correctos (un solo H1).
- [ ] Imágenes con `alt` descriptivos.
- [ ] Schema correcto ([[../03-SEO/SEO-tecnico]]).
- [ ] Al menos 1 link a categoría padre + 2 internos.

## Afiliados
- [ ] Todos los enlaces afiliados pasan por [[../06-Monetizacion/Proteccion-de-enlaces|protección]].
- [ ] No hay enlaces afiliados rotos.
- [ ] `rel="sponsored nofollow"` donde aplique.

## Técnico
- [ ] Indexable.
- [ ] Canonical correcto.
- [ ] OG image presente.
- [ ] Lighthouse / Core Web Vitals OK.
