# 🏠 Homara · Brain

Bienvenido al **cerebro** de Homara. Este vault organiza toda la **estrategia, conocimiento y operaciones** del proyecto. Es un sistema **vivo**: añade, edita y enlaza notas según evoluciona la marca.

> Homara es un **comparador de precios y asistente de compras especializado en hogar**. Web de contenido + comparador + afiliación, pensada para atraer tráfico orgánico, convertir clics y monetizar mediante enlaces de afiliado.

## 🧭 Cómo trabajar aquí

1. Empieza siempre por el **MOC** del área (`_MOC-…`).
2. Usa **`[[wikilinks]]`** entre notas, no rutas.
3. Etiqueta con `#area/...`, `#tipo/...`, `#estado/...`.
4. Crea decisiones en notas atómicas usando [[_Plantillas/Plantilla-Decision|Plantilla-Decision]].
5. Si una nota crece demasiado, **divídela** y enlázala desde su MOC.
