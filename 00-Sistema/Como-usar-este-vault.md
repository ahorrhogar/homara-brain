# Cómo usar este vault

## Filosofía

- **Notas atómicas**: una idea = una nota. Si una nota crece, divídela.
- **MOCs sobre carpetas**: navega por enlaces (`_MOC-…`), no por jerarquía profunda.
- **Frontmatter siempre**: `tags`, `type`, `area`, `estado` cuando aplique.
- **Enlaza generosamente**: cuanto más enlazado, más útil el grafo.

## Flujo típico

1. Tienes una idea, decisión o documento → abre la **plantilla** correspondiente en [[../_Plantillas/_MOC-Plantillas|Plantillas]].
2. Crea la nota dentro del área (`01-…`, `03-…`, etc.).
3. Añádela al **MOC** del área con un wikilink.
4. Etiqueta con `#estado/borrador → #estado/revision → #estado/vigente`.
5. Si referencia otras áreas, enlázalas en una sección "Relacionado".

## Estados de una nota (tag)

- `#estado/idea` — captura rápida.
- `#estado/borrador` — en redacción.
- `#estado/revision` — pendiente de validar.
- `#estado/vigente` — fuente de verdad activa.
- `#estado/archivada` — ya no aplica, se conserva por histórico.
